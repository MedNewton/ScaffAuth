# Product Hunt Draft

## Tagline

Scaffold production-ready Better Auth backends in minutes.

## Description

`create-scaffauth` is an open-source CLI that generates standalone authentication backends using Better Auth.

You choose your framework, database, and ORM, then enable features like OAuth, 2FA, email verification, RBAC, and session tuning.

It can also initialize GitHub repos and deploy to Vercel or Railway.

## Key Features

- 27 stack combinations (3 frameworks x 3 ORMs x 3 databases)
- OAuth provider selection
- 2FA and RBAC toggles
- Email provider support (Resend, SendGrid, SMTP)
- CLI flags for non-interactive workflows

## Call to Action

Try it with:

```bash
npx create-scaffauth
```
